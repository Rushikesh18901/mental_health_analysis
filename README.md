# Mental Health Support Dashboard

An interactive data science dashboard focused on analyzing mental health trends in the technology industry. This project explores key factors influencing employee mental health, visualizes trends using rich graphs, and applies machine learning models to predict mental health outcomes.

## Features

- Interactive visualizations of employee mood and work interference
- Machine learning models to predict mental health treatment needs
- Model performance metrics including accuracy, precision, F1 score, and confusion matrix
- Streamlit-powered web application with clean, professional UI
- Easy-to-use interface for submitting mental health status and getting personalized insights

## Technology Stack

- Python, Pandas, NumPy
- Scikit-learn for machine learning
- Matplotlib & Plotly for visualizations
- Streamlit for dashboard deployment

## Installation and Usage

1. Clone the repository:
   
    git clone https://github.com/Rushikesh18901/mental_health_analysis

    `cd mental-health-dashboard`



2. Install required Python packages:
   
     `pip install -r requirements.txt`


4. Place the `survey.csv` dataset in the `data/` folder.

5. Run the dashboard:
   
    `streamlit run app.py`



## Project Structure

- `app.py` - Main Streamlit app with dashboard and UI logic
- `data/survey.csv` - Primary mental health survey dataset
- `src/` - Scripts for data preprocessing, model building, and utilities
- `requirements.txt` - Python dependencies list
